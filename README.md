# Gameshare


## Run Locally

Clone the project

```bash
  git clone https://github.com/JoaoAfranio/gameshare-front.git
```

Go to the project directory

```bash
  cd gameshare-front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
